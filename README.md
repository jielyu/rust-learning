# Rust学习笔记

rust作为一种内存安全并且语法复杂的语言，被认为是 C++ 的现代版。学习rust语言的主要目的也在于进一步掌握核心性能要求较高的生产模块开发，这一任务之前是以 C/C++ 为主要语言的应用场景。

rust相对于C++的优势主要在内存安全和包管理两方面，但是，跟 C++ 一样入门学习的门槛较高，因此特别记录。
