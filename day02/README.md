# Day 02 猜数字游戏

通过猜数字游戏了解 rust 基本语法和常用工具的使用方式

## 第三方库平台 

官方提供的第三方库托管和查询平台为 [crates.io/](https://crates.io/)。
可以在该平台上寻找需要依赖的第三方库，也可以提交自己开发的库供其他开发者使用。

## 添加依赖库

在项目目录下的 `Cargo.toml` 文件中的 `dependencies` 区域添加依赖库，比如

```
[dependencies]
rand = "0.8.5"
```

## 检查语法

由于cargo编译速度比较慢，为了开发时快速修复语法错误，最好先用快速检查的指令

```bash
cargo check
```

##  代码格式化

提交代码前，最好使用格式化指令

```bash
cargo fmt
```
